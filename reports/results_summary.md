# Résumé des résultats

Ce document regroupe les observations principales :
- Architecture : ReSUnet (U-Net avec blocs résiduels)
- Datasets testés : LiTS2017, LIDC-IDRI (via Kaggle)
- Metrics recommandées : Dice, IoU, précision / rappel.
- Observations : Pré-traiter les volumes (fenêtrage, normalisation) améliore la stabilité.
